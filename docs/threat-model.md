# Threat Model

## Primary Threats

- validator collusion
- false safety or security attestations
- governance capture through concentrated token power
- emergency-pause abuse
- treasury abuse or disclosure gaps
- incident suppression

## Required Controls

- dual-house approval for safety-critical actions
- proposal bonds and timelocks
- slashing and appeal windows
- mandatory public reason codes
- mandatory postmortems after emergency actions

## Current Gaps

- no chain binary yet
- no deterministic validator key workflow yet
- no external audit yet
- no complete legal-review packet yet
